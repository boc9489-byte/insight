# 页面
## 注册页面
- 表单字段：邮箱、验证码、用户名、密码、确认密码
- 点击"获取验证码"按钮发送邮箱验证码（60 秒倒计时）
- 注册成功后跳转到授权页面

## 登录页面
- 表单字段：邮箱、密码
- 登录成功后跳转到授权页面
- 提供"立即注册"和"忘记密码"链接

## 忘记密码页面
- 表单字段：邮箱、验证码、新密码、确认新密码
- 点击"获取验证码"按钮发送邮箱验证码（60 秒倒计时）
- 密码重置成功后跳转到登录页面

## 个人信息页面
布局：侧边栏 + 详细信息

侧边栏包含个人信息、权限管理（有 `*` 权限才显示），最下方是红色登出按钮

个人信息的详细内容展示用户名、邮箱、密码、用户组

用户名、邮箱、密码有修改按钮，点击后展开各自修改框：
- **用户名**：直接输入新用户名并提交
- **邮箱**：需要输入新邮箱和验证码（60 秒倒计时）
- **密码**：需要输入验证码（60 秒倒计时）、新密码、确认密码

提交修改后如果修改成功则收起修改框；密码修改成功后自动登出

点击权限管理跳转到权限管理页面

## 权限管理页面
### 布局
横向平分为【用户】、【组】、【权限】三列，每列高度固定为视口高度，内部独立滚动

列表项：
- 用户栏：ID、用户名、邮箱；编辑与组的关联关系按钮、编辑信息按钮、删除按钮
- 组栏：ID、名称；编辑与用户的关联关系按钮、编辑与权限的关联关系按钮、编辑信息按钮、删除按钮
- 权限栏：ID、名称；编辑与组的关联关系按钮、编辑信息按钮、删除按钮

每栏顶部固定展示【新建】按钮、带【清空】功能的搜索框、以及【字段排序/升降序】切换器

选中项高亮；禁用项透明度降低 (Opacity 50%)，但保留所有交互能力

### 交互
每栏只能选择一个元素

多维交集过滤：
- 选中组，显示此组对应的用户和权限
- 选中用户，显示此用户的组和权限
  - 若再选中某个组，只显示此组对应的权限
  - 若再选中某个权限，只显示包含此权限的组
- 选中权限，显示此权限的组和用户
  - 若再选中某个组，只显示此组对应的用户
  - 若再选中某个用户，只显示包含此用户的组

总结：选中一个元素，获取和选中元素关联的另两栏数据，和现有数据取交集来展示；筛选后无数据时，显示"空"

底部悬浮显示当前筛选的用户、组、权限；点击悬浮按钮取消某个筛选，也可以清空所有筛选

### 关联关系编辑窗口
关联关系编辑窗口：
- 左右分栏展示已关联（左侧）和未关联（右侧）的元素
- 用户展示名称和邮箱；组展示名称；权限展示名称和描述信息
- 左右分栏上方各有一个搜索框
- 点击选中元素
- 左右分栏上方有全选、取消选择按钮
- 左右分栏中间有左右两个切换按钮，点击后将对应侧选中元素移动到对侧
- 点击确定后统一更新关联关系
- 待添加/待移除的项目有绿色/红色边框和标识
